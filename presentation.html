Language improvements move even slower...

  • CSS 3 selectors brought new ways to select elements
  • Discussions on improving the syntax beyond selectors have never made it through the CSS Working Group
  • CSS Variables has been a working draft since 2004 but is only supported in Webkit nightly builds
  • Math functions like calc() have support in Firefox (4+)*, Chrome (19+)* and IE9, and has been a working draft for 11 years!. It's currently at Last Call.

Modular CSS

Re-using CSS has always between projects has always been a challenge, tips and tricks from one project get lost and re-written for another. CSS pre-processors can change that

  • Want a responsive grid and a bunch of gradients? grid.less and mixins.less from Twitter Bootstrap and you're done
  • Sass? Try Mooxins

What's the best thing about SASS?

Keeping my code organized in a way I want to for development and then have it output in the most efficient way for the browser.

Miles Carmany, Web Developer

What do you see as the biggest advantage of using a pre-processor like Scss?

It streamlines the whole process, and is easier to read and debug. It takes only a few minutes to set up but the time savings are immense.

Ghaida Zahran, designer at ZURB

Maintainability and reusability.

Jared Hardy, Product Design at Groupon
