A Brief History of CSS

  • CSS 1 December 1996, separated content from presentation. Basic font properties, colours, alignment, margins, padding and borders
  • CSS 2 May 1998, built on previous spec with new positioning methods and media types
    CSS 2.1 clarified the CSS 2 spec, finally becoming a recommendation as late as June 2011.
  • CSS 3 modularised features forming part of a larger specification. Modules implemented as they're designed by their authors

Improvement has been slow

  • CSS 3 selectors brought new ways to select elements
  • Discussions on improving the syntax beyond selectors have never made it through the CSS Working Group
  • CSS Variables has been a working draft since 2004 and has a small amount of support in Webkit
  • CSS Calculations has basic support in Gecko

Stylesheets have gotten bigger

  • CSS for BBC News website in 2007: 2270 lines in 61.93KB over 12 files
  • CSS for BBC News website in 2012: 19712 lines in 336KB over 12 files over 5 times bigger download and 8 times the number of lines of code
  • CSS for Twitter Bootstrap: 6490 lines in 128KB

Nested rules

CSS


.outer {
	border: 1px solid #999;
}
.outer p {
	padding:0 10px;
	margin: 0 auto;
}
.outer.alert {
	color:red;
}

SASS


.outer {
	border: 1px solid #999;
	p {
		padding: 0 10px;
		margin:0 auto;
	}
	&.alert {
		color:red;
	}
}

Variables

CSS


h1,h2,h3 {
	color:#999;
	text-shadow: 1px 2px 1px #999, -1px -2px 1px #999,
}

SASS


@headingColor: #999;
h1,h2,h3 {
	color:@headingColor;
	text-shadow: 1px 2px 1px @headingColor, -1px -2px 1px @headingColor,
}

SASS


@headingColor: #999;
h1,h2,h3 {
	color:@headingColor;
	text-shadow: 1px 2px 1px @headingColor, -1px -2px 1px @headingColor,
}
